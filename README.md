## ✨ Features

✅ **Interactive Gameplay** - Click to play against computer  
✅ **Score Tracking** - Keep track of wins and losses  
✅ **Visual Feedback** - Clear game results and animations  
✅ **Responsive Design** - Works on desktop and mobile  
✅ **Reset Game** - Start fresh anytime  
✅ **Game Rules** - Clear instructions displayed  
✅ **Random AI** - Computer makes random choices  

---

## 🎮 Game Rules

- **Rock** beats Scissors
- **Scissors** beats Paper  
- **Paper** beats Rock
- Draw if both choose the same

## 🛠️ Technology

- **JavaScript** - Game logic and interactivity

---

## 📊 Scoring System

- Win = +1 point
- Loss = +0 points
- Draw = +0 points

---

## 🎯 Game Flow

1. Choose Rock, Paper, or Scissors
2. Computer makes random choice
3. Compare and determine winner
4. Update score
5. Play again!
